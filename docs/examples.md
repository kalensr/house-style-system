# Examples

Use these examples as patterns for review and revision.

## Contrast Scaffolding

Before:

```text
A checklist can remind me to separate facts from assumptions. It cannot do the thinking for me.
```

After:

```text
A checklist owns repeatable prompts. The writer owns the thinking.
```

## Generic Closing

Before:

```text
This framework paves the way for a brighter future of writing.
```

After:

```text
This framework gives reviewers a place to start.
```

## Weak Evidence

Before:

```text
AI writing is usually worse because it sounds generic.
```

After:

```text
AI-assisted drafts often need review for generic tone, weak evidence, and missing author judgment.
```

## Filler

Before:

```text
It is important to note that the system should be used thoughtfully.
```

After:

```text
Use the system thoughtfully.
```

## Human Review Boundary

Before:

```text
The style gate proves the document is ready.
```

After:

```text
The style gate shows the document avoided known style risks. Human review still owns truth, evidence, and recommendation quality.
```
