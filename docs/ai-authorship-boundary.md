# AI Authorship Boundary

This system is not an AI detector.

It should never be used to prove that a person or model wrote a text.

## Why This Boundary Matters

AI-like writing patterns can show up in human writing. Human writing can also be edited until it looks generic. Style signals are useful for review, but they are weak evidence for authorship.

The safe use is quality improvement:

- clearer claims,
- better evidence,
- stronger judgment,
- better reader fit,
- fewer generic patterns.

## Do Not Use This System To

- accuse someone of using AI,
- score human authorship,
- disguise AI use,
- bypass disclosure rules,
- replace source checking,
- replace human review.

## Use This System To

- improve a draft,
- catch repeated writing risks,
- make review habits visible,
- clarify what automation checks,
- clarify what a human reviewer owns.

## Default Rule

Treat every warning as a review prompt.

Ask: does this wording help the reader trust and use the work?
